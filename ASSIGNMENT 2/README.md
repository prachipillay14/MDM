# 🍽️ Swiggy Clone – Food Delivery Web App

## 📌 Project Overview
This project is a **Swiggy-inspired food delivery web application** built using **HTML, CSS, and JavaScript**.  
It replicates the basic UI and functionality of a real food ordering platform, allowing users to search, filter, and explore food items.

---

## 🚀 Features
- 🔍 Search food items in real-time  
- 🍕 Filter by category (Pizza, Burger, Drinks)  
- ⭐ Filter by rating  
- ⏱️ Sort by delivery time (Fast / Slow)  
- 🎨 Attractive UI with background images  
- 🏷️ Offer badges (like real apps)  
- 🛒 Add to Cart button (UI)  

---

## 🛠️ Technologies Used
- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (ES6)** – Logic & interactivity
